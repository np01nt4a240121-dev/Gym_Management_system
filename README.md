# Gym_Management_system
🏋️ Gym Management System

Java Swing GUI Application for Managing Gym Members

📌 Project Overview

This project is a Gym Management System developed in Java using Java Swing. It allows gym staff to manage Regular and Premium members through a graphical user interface (GUI).

The system demonstrates key Object-Oriented Programming (OOP) concepts such as:

Abstraction

Inheritance

Encapsulation

Polymorphism

Event-driven programming

✨ Features

Regular Member Management

Add new regular members

Activate/deactivate membership

Mark attendance

Upgrade eligibility checking

Revert membership

Premium Member Management

Add premium members

Assign personal trainer

Record payments

Calculate discounts

Track full payment status

GUI Features

User-friendly Swing interface

Separate panels for regular and premium members

Display member information

Clear form functionality

Input validation
